☕ Cafe Menu Project
This project is a simple and elegant Cafe Menu webpage built using HTML and CSS. It presents a clean and organized layout of menu items, including beverages, desserts, and snacks, along with their prices.
The main goal of this project is to practice fundamental front-end development skills such as structuring web pages, applying CSS styling, and creating visually appealing designs. It focuses on readability, layout alignment, and basic responsive design principles.
🚀 Features
Structured menu layout with categories
Clean and modern UI design
Styled using CSS (colors, fonts, spacing)
Easy-to-read format for users
Beginner-friendly code
🛠️ Technologies Used
HTML5
CSS3
Php
📚 What I Learned
How to structure content using HTML
Styling elements with CSS
Working with fonts, colors, and spacing
Creating a visually appealing layout
